# Traveloop

Traveloop is a modern, AI-powered travel itinerary planner and trip management dashboard. 

## Live Demo
🚀 **[View Live Deployment](https://traveloop-lyart.vercel.app)**

---

### Getting Started Locally
1. Clone the repository
2. Install dependencies with `npm install`
3. Set up your `.env` variables for Supabase and Gemini AI
4. Run `npm run dev` to start the development server
